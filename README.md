# 信号与系统实验

## 项目内容
本项目包含4个实验，分别为：

- 波形的合成与分解
- 信号的抽样和内插
- 二阶状态轨迹的显示
- Wav信号的波形分析

这4个实验分别对应本项目中Lab1、Lab2、Lab3、Lab5文件夹，每个文件夹中有对应的README.md文件，这些README.md文件解释了程序的架构，函数的功能以及工程的使用方法。

注意：本项目全部使用的是python语言

## 版本及库依赖
- python版本3.6
- 使用到的库文件在每个Lab文件夹下的README.md文件中有提到
